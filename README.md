# HR-_Attrition_Analytics_Dashboard
HR Attrition Analytics Dashboard build using MySQL, SQL Views and Power BI with Interactive KPI Cards and Business insights.

## Project Overview
This project analyzes employee attrition using MySQL and Power BI. The objective is to identify factors affecting employee turnover and present business insights through an interactive dashboard.

## Tools & Technologies
- MySQL
- SQL
- SQL Views
- DAX
- Power BI
- ODBC Connector
- Microsoft Excel

## Dataset
- Source: IBM HR Analytics Employee Attrition Dataset (downloaded from Kaggle)[HR Attrition Dataset.csv](https://github.com/user-attachments/files/29594036/HR.Attrition.Dataset.csv)

## Dashboard Features
- Total Employees KPI
- Attrition Count KPI
- Attrition Rate KPI
- Attrition by Department
- Attrition by Job Role
- Attrition by Overtime
- Attrition by Gender
- Attrition by Age Group
- Interactive Slicers

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- COUNT()
- CASE WHEN
- Aggregate Functions
- SQL Views

## Key Business Insights
- Research & Development department has the highest attrition.
- Laboratory Technicians show the highest attrition among job roles.
- Employees working overtime have higher attrition.
- Mid-age employees contribute to a significant share of attrition.
